Trust WordPress Theme
======================

A lightweight WordPress/WooCommerce theme based on the Trust eCommerce landing page design.

This theme adapts the landing page's catalogue/product card visual language to a dynamic WooCommerce loop.

Source (landing page): https://github.com/1-karim/trust-ecommerce-landing-page

## Quick notes
- Location: place this folder in `wp-content/themes/` and activate from the WordPress admin.
- Requirements: WordPress 5.8+, WooCommerce plugin.
- Assets: theme assets are in the `assets/` and `src/` folders; build or enqueue as your workflow requires.

## Preview & development
- Activate the theme and visit the Shop archive to see the product catalogue using the adapted markup.
- If you use a build step (Vite), run your usual build/watch to compile `src/` into `assets/`.
